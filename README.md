# 📊✨ Sales Analysis Dashboard (Excel Project)

## 📁 Project Overview
A dynamic and interactive Sales Analysis Dashboard built using 
Microsoft Excel to visualize sales performance, trends, and 
insights in a simple and user-friendly way.

## 🛠 Tools & Technologies Used
- 📊 Microsoft Excel
- 🔄 Pivot Tables & Pivot Charts
- 🎛️ Slicers & Timelines (for interactivity)
- 📐 Excel Formulas (IF, VLOOKUP, SUMIF)

## 🎯 Key Highlights / What I Achieved
- 📈 Built an interactive sales dashboard from raw data
- 🎛️ Used slicers & timelines for dynamic filtering
- 📊 Created meaningful charts for better decision-making
- 🧠 Improved understanding of data visualization & business insights
- 🎨 Learned how to design clean and professional Excel dashboards
