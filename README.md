# 📊✨ Sales Analysis Dashboard (Excel Project)

## 📁 Project Overview
A dynamic and interactive Sales Analysis Dashboard built using 
Microsoft Excel to visualize sales performance, trends, and 
insights in a simple and user-friendly way.

## 🛠 Tools & Technologies Used
- 📊 Microsoft Excel
- 🔄 Pivot Tables & Pivot Charts
- 🎛️ Slicers & Timelines (for interactivity)
- 📐 Excel Formulas (IF, VLOOKUP, SUMIF)
